# Fonts

`DejaVuSans*.ttf` — [DejaVu Sans](https://dejavu-fonts.github.io/) 2.37, embedded in
the generated rulebook PDFs (see `src/data/rulebook-pdf.ts`). Chosen for full Polish
diacritic coverage and the card-suit glyphs (♥ ♦ ♠ ♣) the rulebook needs.
